Estrutura Básica do HTML

  • http://estrutura basica do html

    • Um arquivo HTML é como a "base" de uma página da internet. Ele organiza as informações para que os navegadores saibam como exibi-las. A estrutura básica tem três partes principais:

    • Primeiro item

      • Cabeçalho (head): É onde você coloca informações "invisíveis" para o visitante, como o nome da página (aquele que aparece na aba do navegador) e detalhes para o site funcionar bem.

    • Segundo item

      • Corpo (body): É o que o usuário realmente vê, como textos, imagens, botões e todo o conteúdo da página.

    • Terceiro item

      • Declaração inicial (DOCTYPE html e html): Informa ao navegador que este arquivo é um site e define a linguagem usada (como português).

    O que são Clients?

  • http://o que são clientes

    • Clients : Em programação, clientes geralmente são os usuários ou sistemas que consomem os serviços ou dados de uma aplicação.

    O que são Servers?

  • http://o que são servers?

    • Servers : (Servidores), em resumo, são computadores ou programas que fornecem serviços ou recursos para outros dispositivos ou sistemas, chamados de clientes. Eles "esperam" por pedidos e respondem com os dados ou ações solicitadas..

    Entendendo a diferença entre listas ordenadas e não ordenadas

  • http://entendendo a diferença entre listas ordenadas e não ordenadas

    • A diferença entre listas ordenadas e não ordenadas no HTML está na forma como os itens são exibidos:

    LISTA ORDENADA:

    Os itens têm uma ordem específica.

    Aparecem numerados (1, 2, 3...).

    1. Primeiro
    2. Segundo
    3. Terceiro

    LISTA NÃO ORDENADA:

    Não tem ordem específica.

    Os itens aparecem com marcadores (bolinhas, quadrados, etc.).

    1. Item A
    2. Item B
    3. Item C
